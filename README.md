## 📍 Overview

A digital wedding invitation website for the wedding of **Dr.Vinodh** & **Mamatha**. The wedding will be held on **May 14, 2026 at 11:00 AM** at **KNR GARDENS, Dichpally, Nizamabad, TS**. This website shares event details, family introductions, venue maps, and a countdown to the big day.

---

## 👾 Features

- 💍 Animated wedding invitation with countdown timer
- 🗓️ Full event timeline (Engagement, Haldi, Sangeet, Wedding, Reception)
- 🗺️ Interactive Google Maps for Wedding & Reception venues
- 👨‍👩‍👧‍👦 Family photo gallery with 18 oval photo frames
- 🎵 Background music player
- 🌐 Live visitor counter
- 📱 Fully responsive for mobile and desktop
- ✨ GSAP animations and scroll effects

## 🚀 Getting Started

### ☑️ Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)

### ⚙️ Installation

1. Clone the repository:
```sh
git clone https://github.com/SRIKANTH284/vinodh-mamatha-wedding
```

2. Navigate to the project directory:
```sh
cd vinodh-mamatha-wedding
```

### 🤖 Usage

Simply open `index.html` in your browser — no build step or server required.
